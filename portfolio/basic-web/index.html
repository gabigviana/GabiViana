02.04.01 responsive website

  • Focus

    Building website that adapted and changed its layouts to any device screen size. The first iteration, already built, fit better mobile viewport.

    Art Deco was the given style, for both content and copy.

  • Process

    The initial steps consisted on drawing sketches and determining the breakpoints for the responsiveness. Here, it was possible to gain an idea of what could be achieved and how to plan and carry out the next steps.

    Color scheme, typefaces and interface were communicated through a style tile, which helped form a common visual language of the desirable design. Design principles were applied for enhancing usability and increasing visual appeal.

    The next step was to build a high-fidelity prototype on Adobe XD for larger viewports. With it, it was possible to test the usability and functions on potential users. Some of the test results were applied to the final iteration.

  • Learnings & result

    CSS grid layout and media queries.

    Practical use of design principles.

    High fidelity prototype using Adobe XD.

    User test (with focus on basic UX principles).
